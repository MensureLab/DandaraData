# DandaraData
### O que é o DandaraData ?
DandaraData é um programa faz um cruzamento de dados entre a planilha de comunidades certificadadas pela Fundação Cultural Palmares (FCP) e um dataset contendo artigos relacionados a comunidades quilombolas, o objetivo é descobrir quais comunidades estão sendo citadas nos artigos, relacionando cada artigo com citação explicita a uma comunidade com a comunidade citada juntamente com seus dados contidos na planilha da FCP.
